# go Exercises

A personal repository for practicing and improving my Go (Golang) skills through exercises, small projects, and experiments.



## 📁 Repository Structure

```text
go-exercises/
├── algorithms/
├── basics/
├── concurrency/
├── data-structures/
├── tests/
└── web/
```

Each folder contains small exercises or standalone programs focused on a specific topic.

## 🚀 Running the Projects

Clone the repository:

```bash
git clone https://github.com/emi-gby/go-exercises.git
```

Navigate to a project folder:

```bash
cd basics/frequency-tracker
```

Run the program:

```bash
go run .
```

## 🛠 Requirements

- Go 1.22+ installed
- Basic terminal knowledge

Download Go here:

https://go.dev/dl/
